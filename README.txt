To make this distro:

1. Install Drush (http://drupal.org/project/drush) and Drush Make (http://drupal.org/project/drush_make)
2. Clone this repository using Git. `git clone
git://github.com/KyleAMathews/familyhistorydistro-make-file.git`
3. Make your site by running
`drush make familyhistorydistro.make website_directory`
